Routing de base

Essayez de naviguer sur ces URLs :

  • ./prefix/ : l'URL préfixée statiquement.
  • ./world/ : un résultat similaire aux apps précédentes.
  • ./You/ : on peut changer le nom affiché puisqu'il est extrait de l'URL.
  • ./You/Hey/ : on peut ajouter le paramètre optionnel.

Ici nous utilisons des liens statiques, mais on peut aussi créer les liens dynamiquement. Nous verrons cela dans la prochaine app.
